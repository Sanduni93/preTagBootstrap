To highlight text we user <mark> tag

To underline the text we use <u> tag

To bold the text we use <strong> tag

This specifies use of <pre >tag. Here need to remove unnecessary whitespaces when coding within <pre >tag. If not it will show those line spaces in your web page.

Without whitespaces
<strong> Bold Text </strong>
<u> Underlined text </u>
<mark> Higtlighted text </mark>
With whitespaces and line breakes.
<strong> Bold Text </strong>
                <u> Underlined text </u>
                <mark> Higtlighted text </mark>
